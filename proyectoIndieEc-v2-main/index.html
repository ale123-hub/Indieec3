¿Que es IndieEC?

Indie Ec esta conformado por un grupo de estudiantes de desarrollo de software, el proyecto fue fundado en base a una pasion en este caso la musica.

El proyecto tiene como objetivo crear herramientas digitales a travez de la programacion web, creando secciones para los artistas que deseen participar.

Para participar Contactate con nosotros tienes nuestros datos en la seccion de contactanos.
